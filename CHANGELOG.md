## [0.1.0] - 2026-05-26

### Added

- Added the initial Cion Menu mod setup.
- Added a small Cion Menu button to the title screen and in-game pause menu.
- Added a placeholder menu screen to confirm the button flow works.

### Dependencies

- Requires [cion_core](https://github.com/zero-src/cion_core) `>= 0.1.0`.
- Requires Fabric API on Minecraft 26.2.
